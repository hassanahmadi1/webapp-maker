A simple and ready-to-use template for quickly turning any website into an Android app.

This project is designed so anyone can build their own Android app just by changing a few things like the website URL, app icon, and app name.

Features

Fast and optimized WebView

Smooth loading screen when pages open

Offline cache support (load last cached version if no internet)

Custom offline / error screen

Prefetch internal links for faster browsing

External links open in the system browser

Smart back button handling

Clean and lightweight project

What you need to change

To create your own app, just edit these:

Your website URL in:

MainActivity.kt

App icon in:

res/mipmap

App name in:

res/values/strings.xml

App logo in:

res/drawable/logo.png
Use cases

Online stores

Blogs or news websites

Web apps

Quick Android wrapper for any website

Build

The project can be built locally or using GitHub Actions to generate the APK automatically.
